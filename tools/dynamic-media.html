Dynamic Media

What is Dynamic Media?

Dynamic Media is a service in AEM that allows you to serve optimized, responsive images and videos, dynamically resize and crop assets using URL modifiers, deliver personalized/localized assets, and offload delivery to a high-performance CDN.

Core Capabilities

  • Responsive image and video delivery
  • On-the-fly transformations using URL modifiers
  • Integration with templates and personalization
  • Global CDN-based delivery

Integration with AEM

Dynamic Media is natively integrated into Adobe Experience Manager Assets and can be enabled through configuration. Once set up, it allows seamless publishing, transformation, and delivery of assets directly from within the AEM interface.

Integration Highlights

  • Dynamic Media can be enabled via the AEM Author instance by configuring cloud services
  • Assets are uploaded once and automatically synced to Dynamic Media cloud
  • Publish actions trigger the upload to Dynamic Media servers/CDN
  • Dynamic URLs are generated for each asset and can be used in components
  • Image presets, video profiles, and smart crops are configured within AEM
  • Developers can use Dynamic Media components (e.g., media.html) in editable templates

Image Modifiers (URL Parameters)

Image modifiers are URL query parameters that let you control image properties like size, format, and crop without creating separate renditions.

Common Examples

  • ?wid=300 - Set width to 300px
  • ?hei=200 - Set height to 200px
  • ?fmt=png - Convert to PNG
  • ?crop=100,100,400,400 - Crop to specific coordinates

Video Support

Dynamic Media supports adaptive bitrate video hosting and streaming with custom video player integration.

Key Features

  • Adaptive bitrate streaming
  • Smart encoding
  • Custom video player support
  • Cloud-based delivery

Smart Crop & Responsive Images

Dynamic Media can automatically detect the focal point of an image and generate responsive renditions for different devices.

Benefits

  • AI-based focal point detection
  • Automatic responsive renditions
  • Integration with srcset and lazy loading
  • Improved performance across devices

Dynamic Banners and Image Templates

Use Dynamic Media's image presets and text overlays to create localized or personalized marketing banners.

Capabilities

  • Dynamic background and text swapping
  • Localized campaigns with single template
  • Personalized content delivery
  • Template-driven asset generation

How It Works

Dynamic Media works by processing assets uploaded to AEM and serving them via optimized URLs through a cloud-based image server and CDN.

Process Steps

  • Upload assets to AEM Assets
  • Publish to Dynamic Media cloud
  • Generate URL with modifiers
  • Image server processes request
  • Asset delivered via CDN

Benefits

Dynamic Media offers performance and personalization advantages with automated transformations and global delivery.

Key Advantages

  • Faster page load times
  • Mobile-optimized asset delivery
  • Smart automation like cropping and tagging
  • Personalized and scalable content

Use Case Comparison

Dynamic Media simplifies responsive image delivery by reducing the need to manually create multiple renditions.

Example

  • Without Dynamic Media: Upload multiple image sizes
  • With Dynamic Media: Upload one high-quality image, use URL modifiers or srcset

Troubleshooting

Common issues and fixes while working with Dynamic Media in AEM.

Common Issues

  • Asset not loading: Ensure it is published to Dynamic Media and check CDN URL
  • Image not transforming: Verify URL modifiers
  • Not responsive: Ensure correct use of srcset and image presets
